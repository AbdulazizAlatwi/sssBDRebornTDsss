

## Installation

```sh
# Let's install the bot.
cd $HOME
git clone https://github.com/AbdulazizAlatwi/sssBDRebornTDsss.git
mv sssBDRebornTDsss BDReborn
cd BDReborn
chmod +x beyond.sh
./beyond.sh install
./beyond.sh config
```
